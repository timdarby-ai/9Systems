Frequently Asked Questions

‚ùì How do I access the Python lessons?

Navigate to the Python Programming unit page from the home page. All lesson materials, guides, and video tutorials are available there for download or viewing.

‚ùì What if I'm stuck on my adventure game project?

First, check the Python Quick Reference Guide for syntax help. If you're still stuck, come see me during office hours, send me an email with your specific question, or post in the Google Classroom discussion board where classmates might help too!

‚ùì When is the Python project due?

Project due dates are posted on Google Classroom and announced in class. Generally, you'll have 5-6 weeks to complete your text adventure game after the planning phase.

‚ùì Can I work on my project at home?

Absolutely! Python is free to install. Download it from python.org and use any text editor like IDLE (comes with Python), VS Code, or PyCharm. All project files should be backed up regularly.

‚ùì How can I submit my project?

Projects are submitted through Google Classroom. Make sure your Python file (.py) is named correctly (usually FirstName_LastName_Project.py) and includes all required comments and documentation.

‚ùì Can I get extra help outside of class?

Yes! I'm available during office hours (8:30 AM - 3:30 PM, Monday-Friday) for one-on-one help. You can also email me to set up a specific time if those hours don't work for you.

‚ùì What resources are available for extension students?

Extension students can access advanced topics like object-oriented programming in the Python unit resources. Additional challenge problems and project ideas are available upon request.
